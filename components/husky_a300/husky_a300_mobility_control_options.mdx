### Mobility Control

The Husky can be driven with these interfaces:

1.  [Wireless Joystick](#controller)
2.  ROS's command line interface with the `cmd_vel` ROS 2 topic.
3.  ROS's API, writing your programs in one of the [ROS Client Libraries](https://docs.ros.org/en/rolling/Concepts/Basic/About-Client-Libraries.html), like `rclcpp` or `rclpy`.
4.  Using our [OutdoorNav autonomy API](/docs_outdoornav_user_manual/api/api_overview), which is included in Husky AMP and Husky Observer. 
    You will use a ROS Client Libraries, like `rclcpp` or `rclpy`.
5.  Using our [OutdoorNav web user interface](/docs_outdoornav_user_manual/web_user_interface/ui_overview), which is included with Husky AMP and Husky Observer.
