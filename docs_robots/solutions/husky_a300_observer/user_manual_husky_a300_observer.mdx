The Husky Observer uses a Husky A300 AMP as its base.
The AMP includes the Husky A300 base robot, sensors, autonomy navigation software, and a web user interface for commanding the robot.
Husky Observer then adds Autonomous Wireless Charging, and a Pan Tilt Zoom Camera.

Refer to our [documentation for the base Husky A300](../../outdoor_robots/husky/a300) for understanding the capabilities, safety risks, and use of the base robot.

Also refer to our [documentation for the Husky AMP](../husky_a300_amp/user_manual_husky_a300_amp).


:::safety-danger[Crushing / Impact]

**Never rely on the Husky Observer as the only method to avoid collisions.**

While extensive testing has been performed to validate system robustness, autonomous operation of Husky Observer is not safety rated in accordance with ISO 13849 or similar standards.
The Husky Observer is intended for research and development purposes.
The Husky Observer is not intended for use in deployments where risks are mitigated exclusively through the use of the Husky Observer.
The user is responsible for performing their own Risk Assessment, to ensure the Husky Observer can be operated safely in the user's operating environment.

Contact [Support](#support) for details on how to upgrade to a safety rated autonomy system for Husky A300.

:::
