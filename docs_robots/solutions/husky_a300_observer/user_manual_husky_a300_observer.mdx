The Husky Observer uses a Husky A300 AMP as its base.
The AMP includes the Husky A300 base robot, sensors, autonomy navigation software, and a web user interface for commanding the robot.
Husky Observer then adds Autonomous Wireless Charging, and a Pan Tilt Zoom Camera.

Refer to our [documentation for the base Husky A300](../../outdoor_robots/husky/a300) for understanding the capabilities, safety risks, and use of the base robot.

Also refer to our [documentation for the Husky AMP](../husky_a300_amp/user_manual_husky_a300_amp).
