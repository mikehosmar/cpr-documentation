**Always perform a risk assessment prior to any custom integrations.**

Custom integrations are outside the scope of the robot's safety assessment, as custom integrations may result in new hazards.

:::

<br />

:::note Interfaces

The Husky Observer uses a Husky A300 AMP as its base.
* Husky's electrical interfaces and computer still exist in the base robot.
  Refer to the [Husky A300 Integration page](../../outdoor_robots/husky/a300/integration_husky) page if you want to make use of these interfaces, such as the Husky's GPIO.
* Husky AMP's integration cabinet is available, so you can add custom modules within the IP54 water resistant enclosure.
  Refer to the [Husky AMP Integration page](../husky_a300_amp/integration_husky_a300_amp) page for details.
:::
