The Husky AMP—_(Autonomous  Mobile Platform)_—uses a Husky A300 as its base.
We then then add sensors, autonomy navigation software, and a web user interface for commanding the robot.

Refer to our [documentation for the base Husky A300](../../outdoor_robots/husky/a300) for understanding the capabilities, safety risks, and use of the base robot.


:::safety-danger[Crushing / Impact]

**Never rely on the Husky AMP as the only method to avoid collisions.**

While extensive testing has been performed to validate system robustness, autonomous operation of Husky AMP is not safety rated in accordance with ISO 13849 or similar standards.
The Husky AMP is intended for research and development purposes.
The Husky AMP is not intended for use in deployments where risks are mitigated exclusively through the use of the Husky AMP.
The user is responsible for performing their own Risk Assessment, to ensure the Husky AMP can be operated safely in the user's operating environment.

Contact [Support](#support) for details on how to upgrade to a safety rated autonomy system for Husky A300.

:::
