The Husky AMP—_(Autonomous  Mobile Platform)_—uses a Husky A300 as its base.
We then then add sensors, autonomy navigation software, and a web user interface for commanding the robot.

Refer to our [documentation for the base Husky A300](../../outdoor_robots/husky/a300) for understanding the capabilities, safety risks, and use of the base robot.

## Components Overview {#components-overview}

:::info

The image on this page only shows the new components for the Husky A300 AMP.
Refer to the [Husky A300 Components Overview](../../outdoor_robots/husky/a300/user_manual_husky/#components-overview) for details about the base robot's components.

:::

1.  GNSS antennas and Positioning Unit, _(Fixposition Vision RTK 2)_
2.  Wi-Fi and Bluetooth antennas,
3.  Cellular antennas, all-in-one unit
4.  Teleoperation cameras, _(Luxonis Oak-D Pro W PoE)_
5.  Directional 3D lidar, _(Seyond Robin W)_
6.  Additional Emergency Stop Buttons
7.  Lights
8.  Sounder, _(hidden behind panels)_
