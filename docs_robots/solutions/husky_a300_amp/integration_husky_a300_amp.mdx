The Husky AMP uses a Husky A300 as its base.
Husky's electrical interfaces and computer still exist in the base robot.
Refer to the [Husky A300 Integration page](../../outdoor_robots/husky/a300/integration_husky) page if you want to make use of these interfaces, such as the Husky's GPIO.

:::

### Large Component Mounts, Like Arms / Manipulators {#large-component-mounts}

There is a grid of Ø5 mm holes on the top of the enclosure, near the front and rear edges.
These holes have a pitch of 80 mm, same as Clearpath's PACS system.
The holes are located near the enclosure's weld seams, to improve stiffness between the large component and the Husky's wheels.

### Fusing

The AMP enclosure does not include any new fuses.
Refer to Husky A300's [Electrical Integration](../../outdoor_robots/husky/a300/integration_husky/#electrical-integration) section for fuse locations, and methods to change them.
